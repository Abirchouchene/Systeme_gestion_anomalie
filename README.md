# Système de Gestion des Anomalies

Application basée sur une architecture microservices pour la détection d’anomalies
et la gestion des interventions de maintenance.

## Microservices
- Discovery Server (Eureka)
- Config Server
- API Gateway
- Surveillance Service
- Maintenance Service

## Technologies
- Spring Boot
- Eureka / Feign
- RabbitMQ
- Docker & Docker Compose
